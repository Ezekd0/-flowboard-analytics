## [1.0.0] - 2024-01-01

### Added
- FastAPI backend with versioned REST API routes.
- JWT authentication with access and refresh tokens.
- Task CRUD endpoints and service-layer abstractions.
- Analytics service with metrics/statistics endpoints.
- SQLAlchemy models and Alembic migration scaffolding.
- Redis and Celery integration hooks.
- Docker and NGINX deployment configuration.

### Changed
- Standardized project structure for maintainability.
- Unified environment-based settings across components.

### Fixed
- Validation edge cases for auth and task payload handling.
- Stability improvements in startup configuration defaults.

### Security
- bcrypt password hashing.
- Configurable rate limiting controls.
- Security headers and reverse proxy hardening guidance.
